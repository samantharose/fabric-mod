# Gemstones
### A Fabric mod that adds various gemstones and items made of them to Minecraft

## Requirements
+ Minecraft 1.16.5
+ Fabric 0.11.1
+ Fabric API 0.29.4

## JARs
Grab a JAR from the Releases page or instead build your own JAR by cloning the code.
